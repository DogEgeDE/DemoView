# 费用转账申请审批（方案B）— HTML 可点击原型

## 使用方式
- 打开 `index.html`
- 右上角切换角色：平台运营 / 区县财务 / 市局财务 / 市局出纳
- 如需恢复初始演示数据，点“重置演示数据”

> 数据存储在浏览器 `localStorage`，仅用于原型演示。

## 演示脚本（建议按顺序）

### A. 区县驳回回路
1. 角色切换到【区县财务】→ Tab【待我处理】→ 打开 `TA-COUNTY-0001`
2. 点【区县驳回】→ 角色切换到【平台运营】→ Tab【被驳回】查看该单
3. 在详情页底部“补充资料并再次提交”里修改/补充文件名 → 点【确认再次提交】
4. 再切回【区县财务】→ 该单回到【待我处理】，进入区县重审

### B. 市局驳回回路（区县通过保留）
1. 角色切换到【平台运营】→ Tab【被驳回】→ 打开 `TA-COUNTY-0004`
2. 可看到时间轴：区县节点=通过，市局节点=驳回
3. 补充资料并【确认再次提交】→ 角色切换到【市局财务】→ 该单进入【待我处理】
4. 注意：区县节点仍显示通过（保留不变）

### C. 完整通过到出纳提交网银申请
1. 角色切换到【市局财务】→ 打开 `TA-COUNTY-0003` → 点【市局通过】
2. 角色切换到【市局出纳】→ Tab【待我处理】→ 打开该单 → 点【提交网银转账申请】
3. 该单进入【已完成】（各角色在已完成可见）

### D. 撤回规则验证
1. 角色切换到【平台运营】→ 发起一笔新申请（提交后进入审批中）
2. 在未发生任何审批动作前，详情页显示【撤回】按钮；点击后运营侧状态=已撤回
3. 切换到审批角色：撤回记录不可见
